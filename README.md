# Swing-Graphics-Examples
A set of Java Classes that demonstrate the basics and a little more of Swing Graphics

Purpose is to demonstrate to BEGINNING (first semester, high school) java students how to use Swing Graphics.

## Components used:

1. JFrame
2. JPanel
3. JButton
4. JComboBox
5. JLabel
6. JTextField
7. JRadioButton
8. JScrollPane

## Listeners used:

1. Action Listener
2. Item Listener
3. Key Listener

## Layouts used:
1. BorderLayout
2. BoxLayout
3. CardLayout
4. FlowLayout
5. GridLayout
 
 ## Library methods to assist students
 
 1. getRandom
 2. getRandomColor
